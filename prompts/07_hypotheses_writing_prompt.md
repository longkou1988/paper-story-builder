# 07 Hypotheses Writing Prompt

Task: create `output/hypotheses_outline.md` and `output/hypotheses_draft.md`.

Use only approved paths from `model_spec.json` and `hypothesis_table`.

Path logic:

- Direct effect: explain why IV relates to DV.
- Mediation: explain the process mechanism.
- Moderation: explain the boundary condition and point the moderator to the relationship/path, not only to an outcome variable.
- Chain mediation: explain the ordering logic.
- Moderated mediation: explain the conditional indirect effect.

Do not add unapproved hypotheses. Mark weak evidence as `needs additional literature support`.
