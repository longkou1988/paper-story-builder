# 09 Cross Validation Prompt

Task: create `output/cross_validation_report.md`.

Use this table:

`Check_Item | Status | Evidence | Problem | Revision_Suggestion | Priority`

Statuses: `Pass`, `Minor Issue`, `Major Issue`, `Insufficient Evidence`.

Priorities: `High`, `Medium`, `Low`.

Check consistency among:

- research question
- theory gap
- model
- hypotheses
- variable roles
- contribution claims
- writing claims
- benchmark structure transfer

If inconsistency exists, state the problem before generating fluent prose.
